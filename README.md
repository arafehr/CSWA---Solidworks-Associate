# CSWA Drone Design

This repository contains a 3D CAD model and reference image of a drone assembly created for the CSWA exam/project.

## Project Preview

![Drone Design](CSWA%20Snapshot.png)

## Description

This project shows a quadcopter-style drone design with four propellers, a central body frame, landing supports, and mounted components.
